PostgreSQL to ER diagram

This is a project to expand my knowledge in web development. As a user you can write your SQL code in the textfield down below and a Entity-Relationship Diagram (ER diagram) of your database will appear on the right. This is still a work in progress as you can see. The diagram created lacks much functionality which will be added in the future. Some of my future additions will be functionality for Weak-entities, Custom shapes and an option for the user to upload their own sql file instead of writing in text. The website is purposefully made playful to encurage me to try different elements and styles.
